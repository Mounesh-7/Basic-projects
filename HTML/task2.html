-How to design a favicon using favicon.cc:

-Following are the steps used to design a favicon by using favicon.cc:

  • Visit favicon.cc url in any web browser.
  • Then click on the favicon.ico generator at the favicon.cc web page.
  • There you observe the new empty favicon and to the right of it there will be color picker toolbar.
  • Choosing the required color from the color picker toolbar design the favicon needed.
  • After completing the favicon design you can preview and download the favicon by using Download Favicon option.
  • The downloaded file will be saved in system drive from there you can access if required.
